STEP 1: Push to GitHub
   Command: PowerShell.exe -NoExit -File "push-to-github.ps1"
   What it does:
   ├─ Creates Personal Access Token (you copy-paste)
   ├─ Creates WeamApp repo on GitHub
   └─ Pushes all code to GitHub

STEP 2: Wait for automation
   What happens automatically:
   ├─ Codemagic detects the push
   ├─ Starts building iOS app
   ├─ Starts building Android app
   ├─ Auto-submits to App Store
   └─ Auto-submits to Google Play

STEP 3: Monitor submissions
   What you do:
   ├─ Check App Store Connect for iOS review
   ├─ Check Google Play Console for Android review
   └─ Apps go live once approved
